# Sentiment Analysis Using Transformer Models

### Objective
Built a sentiment analysis model to classify text (reviews, feedback, and general content) as positive or negative, helping understand customer opinions.

### Approach
Fine-tuned the DistilBERT model on the IMDB dataset using Hugging Face Transformers.  
Included preprocessing, tokenization, and model evaluation.  
Developed a Streamlit web app for real-time sentiment classification.

### Outcome
Achieved high accuracy in identifying sentiment from reviews and social media posts.  
Deployed for real-time predictions via Streamlit.

### Tools & Technologies
Python, Hugging Face Transformers, DistilBERT, NLP, Streamlit, PyTorch
